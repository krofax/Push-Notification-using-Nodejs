# Push Notification with Node and Pusher

Example using Node.js and service workers to send and show push notifications

### Version
1.0.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser

```sh
$ npm start
```
